Background

The Horizon Simulation Framework was developed to allow convenient modelling of dynamic Systems using a Breadth First Search algorithm. The software utilizes a brute force approach by evualating all paths between states. While HSF was designed for satellites/space systems, the nature of the algorithm allows for usage in various dynamic systems.
